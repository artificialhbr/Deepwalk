# Deepwalk

Deepwalk是图表示学习算法之一

具体论文请点这里-> http://www.perozzi.net/publications/14_kdd_deepwalk.pdf

## 环境
- gensim==4.1.2
- joblib==1.0.1
- matplotlib==3.4.3
- networkx==2.6.3
- numpy==1.19.5
- scikit_learn==1.0.2

## 使用方法
- pip install -r requirements.txt
- python deepwalk.py
